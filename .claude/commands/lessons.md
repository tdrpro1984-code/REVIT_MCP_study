# /lessons — 智慧提煉指令

從當前對話中提取高階規則或避坑經驗，以 **Append（追加）** 方式寫入 `domain/lessons.md`。

## 執行步驟

1. 回顧本次對話中成功解決的問題或發現的重要規則
2. 提煉為「高階規則」格式（嚴禁只記代碼細節）
3. 指派下一個序號（讀取 `domain/lessons.md` 找最後一條 `[L-XXX]`）
4. 以以下格式 **追加** 到 `domain/lessons.md` 末尾：

```markdown
## [L-XXX] 規則標題

- **規則**：一句話描述核心原則
- **實踐**：具體做法或範例
- **警告**（選填）：常見錯誤或禁止事項
```

## 規則

- 只追加，不修改已有條目
- 必須是可複用的高階規則，而非一次性 debug 紀錄
- 如有多條，逐一追加
