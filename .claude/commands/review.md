# /review — 憲法審計指令

審計 `CLAUDE.md` 是否過於肥大，並提議內容遷移。

## 執行步驟

1. 讀取 `CLAUDE.md` 並統計行數
2. 讀取 `domain/` 目錄下所有檔案列表
3. 識別 `CLAUDE.md` 中屬於「具體規格或案例」而非「高階規則」的段落
4. 輸出審計報告：

```
## /review 審計報告

- CLAUDE.md 目前行數：XXX 行
- 建議遷移項目：

| 段落 | 建議遷移至 | 原因 |
|------|-----------|------|
| ... | domain/xxx.md | 屬於具體流程，非架構說明 |
```

5. 詢問使用者確認後執行遷移

## 觸發條件

- `CLAUDE.md` 超過 150 行時主動建議執行 `/review`
- 使用者明確呼叫時執行
