# 第一次安裝遇到問題？照這份走就好

> 這份文件是給第一次安裝、遇到紅字錯誤的同學。
> 不需要任何程式背景，照著一步一步做就好。

---

## 你遇到的問題是什麼？

### 問題一：貼指令時出現「'&&' 語彙基元不是有效的陳述式分隔符號」

**白話解釋：**
Windows 有兩種黑色視窗（終端機），它們長得很像但規則不同。
我們給的指令用了 `&&` 把兩個動作接在一起，但你開的那種黑色視窗不認識 `&&`。
這就像用日文文法寫了一句話，但對方只看得懂中文——不是你的問題，是我們給的指令格式不對。

**我們已經修正了。** 現在網頁上的指令已經拆成兩行，不會再遇到這個問題。

### 問題二：跑 setup.bat 時出現一整片紅字

**白話解釋：**
安裝腳本裡面有寫中文說明（例如「正在安裝 Node.js…」）。
但 Windows 預設的讀取方式看不懂這種中文編碼格式，
所以它把每個中文字都讀成亂碼，然後因為亂碼不是合法的程式指令，就整個壞掉了。
這就像你打開一個 Word 檔案，但電腦用記事本開——內容全變亂碼。

**我們已經修正了。** 所有安裝腳本都加上了正確的編碼標記，Windows 現在可以正確讀取中文。

## 重新安裝步驟（從頭來）

### 第一步：刪除之前下載的資料夾

因為之前下載的版本有問題，我們要重新下載一份修正後的版本。

1. 打開你的**桌面**
2. 找到資料夾 **REVIT_MCP_study**（如果有的話）
3. **右鍵 → 刪除**（或拖到資源回收桶）

> 如果桌面上沒有這個資料夾，可能在其他地方。沒關係，跳過這步。

---

### 第二步：打開終端機

1. 在鍵盤上同時按 **Win 鍵 + X**
   - Win 鍵就是鍵盤左下角有 Windows 標誌的那個鍵
2. 會彈出一個選單，請點選：
   - 如果看到「**終端機**」→ 點它
   - 如果看到「**Windows PowerShell**」→ 點它
   - 如果兩個都有 → 點「**終端機**」

> 你會看到一個黑色（或深藍色）的視窗，裡面有一行文字閃著游標。
> 這就是「終端機」。接下來我們只需要在這裡貼指令就好。

---

### 第三步：切換到桌面

在終端機裡面，**複製以下這一行**，貼上去，然後按 **Enter**：

```
cd Desktop
```

> **怎麼貼？** 在黑色視窗裡面，用 **右鍵點一下** 就是貼上（不是 Ctrl+V）。
> 有些版本可以用 Ctrl+V，試試看哪個有效。

貼完按 Enter 後，你應該會看到畫面上的路徑從類似 `C:\Users\你的名字>` 變成 `C:\Users\你的名字\Desktop>`。

**如果出現紅字「找不到路徑」：**
你的電腦可能用中文桌面路徑，改貼這一行試試：
```
cd 桌面
```

---

### 第四步：下載專案

在終端機裡面，**複製以下這一行**，貼上去，按 **Enter**：

```
git clone https://github.com/shuotao/REVIT_MCP_study.git
```

你會看到畫面開始跑一些數字（下載進度）。等它跑完，出現新的一行 `>` 提示符號，就是下載完成了。

**如果出現「'git' 不是內部或外部命令」：**
你的電腦還沒裝 Git。請做以下步驟：
1. 打開瀏覽器，前往 https://git-scm.com/download/win
2. 它會自動開始下載安裝檔
3. 打開安裝檔，**一路按 Next**，最後按 Install
4. 安裝完成後，**關掉黑色視窗**
5. 重新從「第二步」開始做（重新開終端機）

---

### 第五步：跑安裝程式

1. 打開你的**桌面**
2. 打開資料夾 **REVIT_MCP_study**
3. 打開裡面的 **scripts** 資料夾
4. 找到 **setup.bat**
5. **雙擊它**

> 會跳出一個黑色視窗，開始自動安裝。
> 過程中可能會問你問題（例如選擇 Revit 版本），照畫面上的提示回答就好。
> 整個過程大約 3-5 分鐘。

**如果又出現紅字：**
請確認你是在「第四步」下載的新版本（不是之前的舊版本）。
舊版本的安裝檔有中文編碼問題，新版本已經修正。
最簡單的確認方式：
1. 回到終端機
2. 貼上以下指令，按 Enter：
```
cd Desktop\REVIT_MCP_study
```
3. 再貼上：
```
git log --oneline -1
```
4. 如果看到的訊息裡有「**UTF-8 BOM**」或「**修復 Windows 中文亂碼**」字樣，就是新版本
5. 如果不是，貼上以下指令更新：
```
git pull
```
6. 更新完後，再回去雙擊 setup.bat

---

### 第六步：確認安裝成功

安裝程式跑完後，畫面最後會顯示一個總結：
- 如果看到綠色的 `[OK]` 訊息 → 安裝成功
- 如果看到紅色的 `[!!]` 訊息 → 那一步失敗了（看訊息內容判斷）

**安裝成功後，你需要做的最後一件事：**
1. 打開 Revit
2. 在上方找到 **Add-in** 或 **增益集** 頁籤
3. 找到 **RevitMCP** 按鈕
4. 點一下啟動它

看到 RevitMCP 按鈕亮起來就完成了！

## 常見問題

### Q：我的桌面叫「桌面」不是「Desktop」，指令怎麼打？
用 `cd 桌面` 取代 `cd Desktop`。

### Q：我按 Win+X 沒有看到「終端機」或「PowerShell」？
試試在桌面空白處 **右鍵** → 看有沒有「在終端機中開啟」或「在這裡開啟 PowerShell」。

### Q：下載到一半斷掉了怎麼辦？
刪掉桌面上的 REVIT_MCP_study 資料夾，重新從第四步開始。

### Q：setup.bat 問我選 Revit 版本，我不知道選什麼？
選你電腦上有安裝的 Revit 版本。例如你用的是 Revit 2024，就輸入 `2024`。
如果不確定，打開 Revit 看一下標題列上寫的版本號。

### Q：我做到最後一步但 Revit 裡面沒有看到 RevitMCP 按鈕？
可能是 DLL 沒有部署成功。請在終端機裡貼上：
```
cd Desktop\REVIT_MCP_study
powershell -ExecutionPolicy Bypass -File scripts\install-addon.ps1
```
然後重新開啟 Revit。
